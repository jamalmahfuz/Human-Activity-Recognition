# LSTM for Human Activity Recognition

Human activity recognition using smartphones sata set and an LSTM ANN. Classifying the type of movement amongst six categories:
- WALKING,
- WALKING_UPSTAIRS,
- WALKING_DOWNSTAIRS,
- SITTING,
- STANDING,
- LAYING.
